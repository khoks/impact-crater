## Description

Two coupled decisions:

1. **Music modes.** User input (this session): two modes are in scope.
   - *Standard mode* — selected/generated background music plays under the curated video.
   - *Music-video mode* — the result is synced as a music video around the user's supplied music + theme; user can describe in natural language which sections of the music should be built from which media.

2. **MVP music sourcing.** Where does music come from at MVP? User-supplied only? Royalty-free starter pack? Licensed library integration?

Recommendation to land: user-supplied only for MVP; small royalty-free starter pack and licensed-library integration → v1.
