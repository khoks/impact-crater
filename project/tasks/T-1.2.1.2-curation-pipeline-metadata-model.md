## Description

Lock the *shape* (not the implementation) of the curation pipeline and the metadata model attached to every photo and video scene.

User input (this session): hybrid pipeline (deterministic pre-filter → multimodal-LLM judgment + ordering) is the agreed shape, with rich per-media metadata tagging on top.

Per-photo metadata to capture: time of day, people in focus + identities, lat/long + location description, timestamp, mood, lighting, quality, foreground + background activity, visible objects (S/M/L), clothing, pose-quality scores across multiple categories, plus generic + user-task-context-specific tags.

Per-video metadata: scene-segmented first; each scene gets its own metadata as above. File type, codec, size, duration also tagged.

Cross-job analysis reuse via a content-addressed cache (so the same photo is not re-analyzed across jobs).
