## Description

Lock where projects, source media, derived artifacts, embeddings, the cross-job cache, and the audit log live on disk; whether source media is copied or referenced; what database (if any) holds metadata. Foundational for A-001 (project model), A-003 (audit log), A-005 (failure recovery), A-010 (stable IDs), A-011 (cache), N-003 (versioned artifact), N-007 (cache schema).

User accepted: **per-project tree under `~/.impact-crater/projects/`; SQLite for metadata; source media referenced (path + content_hash), not copied; snapshot directories under `snapshots/` per N-003; cross-project cache content-hash + model-version keyed under `~/.impact-crater/cache/`; audit log as append-only JSONL.**
