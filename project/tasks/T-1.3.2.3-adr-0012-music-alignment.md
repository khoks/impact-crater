## Description

Lock the music analysis + alignment pipeline that turns user-supplied audio into a beat-grid + section structure + render-time alignment. Two modes per D-010: standard (background music) + music-video (cuts driven by music). Music sourcing = user-supplied at MVP per D-018.

User redirects (2026-05-02):
- Music structure analysis = **Madmom** (state-of-the-art beat / downbeat detection); section detection via librosa.segment (Madmom is beat-focused, not section-focused)
