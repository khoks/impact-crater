## Description

Lock the deterministic media-handling layer that runs *before* any LLM sees media: photo decoding (JPEG/PNG/WebP/AVIF/HEIC/RAW), video decoding (ffmpeg), perceptual hashing for dedup, scene segmentation (PySceneDetect), face detection, smart-crop, aspect-ratio handling, and where rendering happens.

User redirects (2026-05-02):
- HEIC + RAW supported at MVP
- Dedup off-by-default with surfaced suggestion
- Vision-LLM-only face detection + a novel **person-library + reference-collage** recognition mechanism (filed as N-008): build a per-person library of N face photos, construct a labeled reference-collage at recognition time, the vision LLM identifies matches in new photos via a structured-output schema
- Smart-crop = `smartcrop.py` saliency CNN
